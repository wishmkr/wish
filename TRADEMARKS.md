# WishOS Trademark Policy

## Official project marks

The names **Wish**, **WishOS**, **Wish Package Manager**, their logos,
and their official visual branding identify the official WishOS project.

The official project is administered by:

- Primary project owner: **millieleonie**
- Secondary project owner: **wishmkr**

The GNU Affero General Public License applies to the source code.
It does not grant permission to present a modified or unofficial version
as an official WishOS release.

## Forks and modified versions

Forks and modified versions are permitted under the software license.

A modified version must:

1. Clearly state that it is modified and unofficial.
2. Clearly state that it is based on, derived from, or forked from WishOS.
3. Link visibly to the original WishOS repository.
4. Preserve applicable copyright and license notices.
5. Use a different primary product name and logo.
6. Not imply endorsement, approval, sponsorship, or official status from
   the WishOS project or its owners.

A suitable notice is:

> This project is an independent fork of WishOS.  
> Original project: https://github.com/wishmkr/wishos  
> This project is not an official WishOS release and is not endorsed by
> the WishOS project.

## Permitted descriptive use

Truthful descriptive statements such as the following are permitted:

- “Based on WishOS”
- “Forked from WishOS”
- “Derived from WishOS”
- “Compatible with WishOS”

Such use must not suggest that the modified project is an official
WishOS product.

## Unmodified redistribution

An exact, unmodified copy of an official WishOS release may retain the
WishOS name and branding when all copyright and license notices remain
intact and the release is accurately represented as an official,
unmodified release.

## Restricted uses

Without written permission from the project owners, modified or
unofficial versions may not:

- use Wish, WishOS, or Wish Package Manager as their primary product name;
- use the official WishOS logo as their main logo;
- claim to be maintained, certified, approved, or endorsed by the
  official WishOS project;
- use branding intended to impersonate or cause confusion with the
  official project.
